# Running RancherOS

RancherOSは、仮想化プラットフォーム、クラウドプロバイダー、およびベアメタルサーバー上で動作します。
私たちはあなたのラップトップ上でローカルVMを実行することもサポートします。
できるだけ早くRancherOSの実行を開始するには、[クイックスタートガイド](https://rancher.com/docs/os/v1.x/en/quick-start-guide/)に従ってください。
